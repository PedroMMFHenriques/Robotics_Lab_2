run the program with the command:
python navigation.py

upon running the program, it will ask for initial energy for the car in Watt hour (Wh). 10 is sufficient for any path.
in the top left corner, the satelite indicates if the gps signal is lost and the battery indicates how low the car energy is.
demonstration of the program working: https://youtu.be/WYmAbRX0UQc

pip install numpy
pip install scipy

pip install pygame

pip install pyqt5
